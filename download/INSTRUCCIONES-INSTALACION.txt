================================================
CHIACCHIO - GUIA DE DESCARGA E INSTALACION LOCAL
================================================

Este archivo contiene todas las instrucciones para
configurar el proyecto en tu PC con XAMPP.

================================================
PASO 1: REQUISITOS PREVIOS
================================================

Asegurate de tener instalado:
1. Node.js LTS (v18 o superior) - https://nodejs.org
2. XAMPP con MySQL - https://www.apachefriends.org
3. VS Code - https://code.visualstudio.com
4. Git (opcional) - https://git-scm.com

Para verificar Node.js:
- Abrir CMD o PowerShell
- Escribir: node --version
- Debe mostrar v18.x.x o superior

================================================
PASO 5: INSTALAR DEPENDENCIAS
================================================

1. Abrir VS Code
2. Archivo > Abrir Carpeta > C:\xampp\htdocs\chiacchio
3. Abrir terminal integrada (Ctrl + ñ)
4. Ejecutar:

npm install

Esto instalara todas las dependencias (puede tardar 2-3 minutos)

================================================
PASO 6: EJECUTAR EL PROYECTO
================================================

En la terminal de VS Code:

npm run dev

Luego abrir en el navegador:
http://localhost:3000

================================================
PASO 7: CONFIGURAR XAMPP (FASE 2)
================================================

1. Abrir XAMPP Control Panel
2. Iniciar Apache y MySQL
3. Abrir phpMyAdmin: http://localhost/phpmyadmin
4. Crear nueva base de datos llamada "chiacchio"
5. Cotejamiento: utf8mb4_general_ci

================================================
NOTAS IMPORTANTES
================================================

- El proyecto usa puerto 3000 por defecto
- No uses puerto 3000 para XAMPP (Apache usa 80, MySQL usa 3306)
- Los datos mock se guardan en localStorage
- Para FASE 2 conectaremos con MySQL
